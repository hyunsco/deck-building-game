# Slay the Spire 메커니즘 조사 (1막 / 아이언클래드 중심)

> 본 게임 제작을 위해 공개 위키(slay-the-spire.fandom.com, slaythespire.wiki.gg 등)에서
> 수집한 **규칙·수치 데이터** 요약입니다. 게임 규칙과 수치는 사실 정보이며, 원작의 문장·아트는 포함하지 않습니다.
> 본 프로젝트 구현과 다른 부분은 ⚠️로 표시.

## 핵심 전투 규칙

- 에너지 매 턴 **3**, 미사용분 소멸. 카드 비용 0~3 또는 X(남은 에너지 전부).
- 턴 시작 시 **5장** 드로우, 손패 최대 10장, 턴 종료 시 손 전체 버림 (유지/휘발성 예외).
- 뽑을 더미가 비면 버린 더미를 섞어 재사용.
- **방어도(Block)**: 공격 피해를 흡수, **자기 턴 시작 시 소멸**. 적도 동일.
- HP는 전투 간 유지(자동 회복 없음). 아이언클래드 시작 유물이 전투 후 6 회복.
- **소멸(Exhaust)**: 해당 전투에서 카드 제거. 소멸 트리거(고통 무시 등) 발동.
- 카드 타입: 공격 / 스킬 / 파워(전투 내 영구 효과) / 상태이상(전투 후 소멸) / 저주(덱에 잔류).

## 상태 효과 수치

| 효과 | 수치 | 비고 |
|---|---|---|
| 취약 Vulnerable | 받는 공격 피해 **+50% (내림)** | 턴 수로 감소 |
| 약화 Weak | 주는 공격 피해 **-25% (내림)** | 턴 수로 감소 |
| 손상 Frail | 카드 방어도 **-25% (내림)** | 턴 수로 감소 |
| 힘 Strength | 타격당 +1 (다단히트 전부 적용) | 전투 내 영구, 음수 가능 |
| 민첩 Dexterity | 카드 방어도 +1 | 전투 내 영구 |

계산 순서: (기본 + 힘) → 약화 ×0.75 → 취약 ×1.5, 각 단계 내림.

### 정예

- **도깨비 거두** 82–86: 첫턴 격분 2(스킬 사용마다 힘+2) → 돌진 14 / 두개골 강타 6+취약2
- **잠자는 갑충** 109–111: 수면+금속화 8, 피해 입으면 기상(1턴 기절), 기상 후 18·18·영혼 흡수(힘·민첩 -1)
- **파수꾼 ×3** 38–42: 결계 1, 몽롱함 2장 섞기 ↔ 광선 9 교대 (1·3번은 볼트 선공)

### 보스

- **거대 수호자** 240: 모드 전환 30(피해 누적 시 방어 태세 + 방어도 20, 임계값 +10씩).
  공세: 충전(방9) → 강타 32 → 증기(약화2·취약2) → 회전 5×4. 수세: 가시 외피 3 → 9 → 8×2 후 복귀
- **육염령** 250: 1턴 대기 → **영혼 가르기 6×(현재HP/12+1)** → 지지기(6+화상)·박치기(5×2)·점화(힘2·방12)·**업화(2×6+화상3, 화상 강화)** 7턴 주기
- **슬라임 대왕** 140: 점액 3장 → 꿈틀 → **짓뭉개기 35** 반복, 50% 이하에서 대형 산성+가시 슬라임으로 **분열**

## 런 구조

- 1막 = 15층 + 보스. 지도 **7열 × 15행**, 6개 경로 랜덤워크, 교차 없음.
- 고정: 1층 몬스터(쉬움 풀) · **9층 보물** · **15층 휴식처**. 정예/휴식처는 6층 이전 금지, 14층 휴식처 금지.
- 방 배정 비율(원작): 몬스터 53% / ? 22% / 휴식 12% / 정예 8% / 상인 5% ⚠️ 본 구현은 45/22/16/12/5로 정예를 약간 상향.
- 처음 3번의 일반 전투는 쉬움 풀(광신도/턱벌레/유충2/소형 슬라임)에서 출현.
- 휴식처: **휴식 = 최대 체력 30% 회복** / 단련 = 카드 1장 영구 강화.
- 보상: 금화(일반 10–20, 정예 25–35), 카드 3택1(일반 60/37/3 · 정예 50/40/10), 물약 40%±10%, 정예는 유물 추가.
- 시작: HP **80**, 금화 **99**, 덱 = 타격×5 + 수비×4 + 강타×1, 유물 = 끓는 피.
