# Slay the Spire UI/UX 디자인 조사 → 구현 매핑

> 원작의 화면 구성·디자인 언어를 조사하고, **오리지널 CSS/SVG**로 같은 룩앤필을 재현하기 위한 설계 문서.
> 원작 아트 에셋은 일절 사용하지 않음. 기준 해상도 1920×1080 고정 스테이지 + `transform: scale()` 맞춤.

## 전투 화면 레이아웃 (원작 → 본 구현)

```
┌──────────────────────────────────────────────────────────┐
│ 상단바: 캐릭터 · ❤HP · 🪙골드 · 물약 3슬롯 · 층수 · 덱     │
│ 유물 줄: ◉ ◉ ◉ …                                         │
│                                  [의도 ⚔12]   [의도 🛡]   │
│   [플레이어]                      [적1]      [적2]        │
│   HP바+방어도 뱃지+상태칩          HP바…       HP바…       │
│                                                          │
│ ⬤3/3   ╱🂠🂠🂠🂠🂠╲ (부채꼴 핸드)         [턴 종료]         │
│ [뽑을 24]                        [소멸 2][버린 11]        │
└──────────────────────────────────────────────────────────┘
```

- **의도 시스템**: 적 머리 위에 검(피해량 표기, 20+면 대형 검) / 방패 / ↑버프 / ↓디버프 / 카드(상태이상 섞기) /
  Zzz / ? 아이콘. 피해량은 힘·약화·취약 보정 후의 실제 예상값 표시 — 원작과 동일.
- **HP바**: 검붉은 트랙 + 붉은 채움 + 흰 숫자. 방어도가 있으면 좌측에 청회색 방패 뱃지 + 바 테두리 푸른빛.
- **카드 부채꼴**: 카드 i의 회전 = `(i-(n-1)/2)×clamp(40/n,3,7)°`, 끝일수록 아래로. 호버 시 직립+1.32배 확대.
  사용 가능 카드는 **초록 펄스 글로우**, 에너지 부족 시 비용 숫자 빨강.
- **타게팅**: 공격 카드 선택 → 빨간 점선 베지어 화살표가 커서를 따라감 → 적 클릭으로 확정 (원작의 드래그를 클릭 2회로 변형).
- **에너지 구슬**: 좌하단 호박색 라디얼 그라디언트 구슬, 금테, 3/3 표기. 0이면 채도 감소.

## 카드 디자인

- 세로 5:7 비율 (200×280). 좌상단 **비용 구슬**(호박색), 상단 이름 배너, 중단 아트 창, 타입 리본, 하단 설명.
- **타입별 프레임 형태**(원작의 시각 언어): 공격 = 아트 창 하단이 뾰족한 오각형(clip-path), 스킬 = 사각형, 파워 = 타원.
- **희귀도 = 테두리 색**: 일반 회색 `#9aa0a6` / 특별 청록 `#5fb6c9` / 희귀 금색 `#d4af37`.
- 강화 카드는 이름이 **초록색 + "+"**. 키워드(취약·약화·소멸…)는 금색 강조, 버프로 수치가 변하면 초록 숫자.

## 지도 화면

- 세로 스크롤 **양피지** 시트(밝은 갈색 그라디언트 + 안쪽 그림자 비네팅), 7열×15층 노드.
- 경로는 갈색 **점선 스티치**, 지나온 길은 검붉은색. 노드는 잉크 스탬프풍 SVG 아이콘
  (몬스터=교차 검, 정예=화염, 휴식=모닥불, 상인=동전, 보물=상자, ?=물음표, 보스=뿔 달린 해골).
- 노드마다 ±14px 지터 + ±6° 회전으로 손그림 느낌. 현재 위치는 붉은 링 펄스, 선택 가능 노드는 초록 점선 링.
- 방문한 노드는 붉은 ✕ 도장.

## 색상 팔레트 (CSS 변수)

| 역할 | 값 |
|---|---|
| 배경 | `#0b0a0c` ~ `#17120e` (웜 블랙) |
| 프레임 목재/석재 | `#2a1f14` `#3b2a18` `#5a4226` |
| 금색 3단계 | `#8a6d3b` → `#c9a45c` → `#f5d76e` |
| 양피지 | `#e8d9b0` `#d9c79a` `#b59a6a` |
| HP | 채움 `#b3271e` / 트랙 `#3a0d0a` |
| 방어도 | `#9fb8c8` |
| 에너지 | `#e8a33d` |
| 희귀도 | `#9aa0a6` / `#5fb6c9` / `#d4af37` |

원칙: 배경은 어둡고 탈색되게, 채도 높은 색은 **정보**(HP·에너지·의도·글로우·희귀도)에만 사용.

## 타이포그래피

- 원작 영문 UI 폰트는 **Kreon**(슬랩 세리프, OFL 무료) — 숫자/라틴에 그대로 사용.
- 한글은 **Noto Serif KR**로 판타지 세리프 느낌을 맞춤. 숫자는 굵게 + 어두운 텍스트 섀도(외곽선 효과).
- 타이틀 로고는 금색 그라디언트 + 드롭 섀도 베벨로 원작의 육중한 금속 로고 느낌만 재현 (직접 트레이싱 없음).

## 연출 (게임필)

| 연출 | 구현 |
|---|---|
| 피해 숫자 | 타격 지점에서 떠오르며 페이드 (1초), 방어 흡수 시 회색 |
| 피격 | 흰색 플래시 + 좌우 흔들림 (`hitFlash` 키프레임) |
| 큰 피해 | 스테이지 전체 셰이크 (±7px/±14px 감쇠) |
| 적 행동 | 행동명 플로팅 + 런지(앞으로 70px 돌진 후 복귀) |
| 사망 | 채도 제거 + 페이드 + 하강 |
| 턴 배너 | "적의 턴 / 나의 턴" 중앙 확대 페이드 |
| 대기 호흡 | 스프라이트 상하 5px 바운스, 의도 아이콘 부유, 글로우 펄스 — 화면이 완전히 정지하지 않게 |
| 휴식처 | 모닥불 SVG 불꽃 모핑 + 주황 라디얼 글로우 플리커 |
